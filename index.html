Doctype

Se utiliza en documentos HTML para especificar la versión de HTML o XHTML que se está utilizando en el documento. Su propósito principal es informar al navegador web y otros agentes de usuario sobre el tipo de documento que están a punto de procesar, lo que asegura que el documento se renderice y se interprete correctamente.

Head

Se utiliza para incluir metadatos y otros elementos que no se muestran directamente en la página web, pero que son esenciales para su funcionamiento, presentación y optimización. La sección head se encuentra dentro del documento HTML y precede a la sección body, que contiene el contenido visible de la página.


El Body

Representa el contenido principal de un documento HTML y contiene todo lo que se mostrará en la ventana del navegador web. En otras palabras, el contenido que coloques dentro de body es lo que los usuarios verán en la página web cuando la visiten. En resumen, el elemento body en HTML es esencial ya que contiene todo el contenido visible y la estructura de una página web, permitiéndote crear páginas web interactivas y atractivas para los usuarios.

El Main

Se utiliza para marcar el contenido principal de una página web. Es decir, el contenido que es esencial para entender o interactuar con la página. La etiqueta main ayuda a los motores de búsqueda y a las tecnologías de asistencia a identificar fácilmente qué parte del contenido es la parte principal o central de la página.

Header

Se utiliza para definir la sección de encabezado de un documento o de una sección dentro de un documento. Esta sección generalmente contiene elementos como el título de la página, logotipos, menús de navegación y otra información importante

El Div

Es uno de los elementos más versátiles en HTML y se utiliza principalmente como un contenedor genérico para agrupar y estructurar elementos y contenido dentro de una página web. No aporta ningún significado semántico en sí mismo, pero es esencial para crear diseños y estructuras complejas en HTML y para aplicar estilos CSS a grupos de elementos.

Aside:

Esta etiqueta se usa para mostrar un contenido secundario que se muestra en el lado derecho o izquierdo de la página, dependiendo del diseño.

Img

Se utiliza en HTML para insertar imágenes en una página web. Esta etiqueta es esencial para mostrar gráficos, fotografías, logotipos y otros elementos visuales en una página web.

Link

Se utiliza para establecer relaciones entre el documento actual y recursos externos. Aunque es más conocida por vincular hojas de estilo (CSS) a documentos HTML, su función principal es definir relaciones entre el documento actual y otros recursos.
